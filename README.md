## Project Status

This project is currently paused until duplicate removal through the Spotify API can be confirmed to be safe and reliable.

Duplicate detection still works. The app can scan your playlists, identify duplicate tracks, and show the playlist positions of duplicate entries.

Removal is intentionally disabled for now. If you want to clean up a playlist, use the displayed playlist positions to manually remove duplicates in Spotify.

## Getting Started

```bash

# or
```
